# it-tools

> 旨在帮助程序员快速开发的小工具.
> 让生活更加美好.

这个工具目前包括“正则表达式”、“时间戳转化”、“颜色盒子”、“Json转化”、“Base64转化“、“定时任务工具”


### 特点

- 正则表达式
- 时间戳转化
- 颜色盒子
- JSON 转化
- Base64 转化
- 定时任务

### 构建步骤

``` bash
# 安装依赖
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# run unit & end-to-end tests
npm test


# lint all JS/Vue component files in `src/`
npm run lint

```
